# 人生困惑 / 情绪整理

这个目录用于记录我在人生、工作、关系、自我认知中的困惑与情绪波动。

我建立这个目录，不是为了追求“立刻释怀”，而是为了：

* 整理混乱
* 区分事实与情绪
* 看见长期重复的问题
* 在清醒中保留行动力
* 为未来复盘留下真实材料

## 适用内容

这里主要记录：

* 情绪波动
* 人际关系困扰
* 工作中的委屈、压抑、失望、无力感
* 决策犹豫与内耗
* 阶段性的迷茫、自我怀疑、价值感波动
* 一些暂时无法解决，但值得认真记录的问题

## 使用方式

当我陷入内耗、情绪波动、关系困扰或阶段性迷茫时，优先记录：

1. 发生了什么
2. 我有什么感受
3. 我在如何解释这件事
4. 哪些是事实，哪些是投射
5. 当前真正的问题是什么
6. 我现在能做的最小动作是什么

## 单篇记录模板

### 1. 起因

发生了什么事情。

### 2. 当下情绪

我此刻最明显的情绪是什么。

### 3. 我脑中的解释

我现在是怎么理解这件事的。
我在担心什么。
我默认了什么前提。

### 4. 已知事实

哪些是可以确认的事实。
哪些只是我的猜测、联想、投射、放大。

### 5. 真正刺痛我的点

这件事为什么会让我这么难受。
它触发了我什么旧问题。

### 6. 更可能真实的情况

如果不被当下情绪完全带走，更接近现实的理解可能是什么。

### 7. 我能做什么

现在最小、最现实的动作是什么。

### 8. 这次记录的结论

用一句话总结这次整理。

## 阶段复盘方向

之后可以定期回看：

* 最近反复出现的情绪是什么
* 哪些问题已经改善
* 哪些模式还在重复
* 哪些关系或环境正在持续消耗我
* 我是否比以前更能识别自己的情绪和边界

## 记录原则

* 允许复杂，不强行正能量
* 不用空泛安慰代替分析
* 先澄清，再判断
* 理解别人，不等于取消边界
* 目标不是完美，而是更诚实地看清自己

## 命名建议

单篇记录尽量统一格式：

`YYYY-MM-DD-标题.md`

例如：

`2026-03-29-理解别人之后的失望与身不由己.md`
