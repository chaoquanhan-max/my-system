# 我的个人系统

## 基本信息
- 职业：赌场经理
- 工作内容：赌场运营与管理

## 目标
- 控制体脂 + 提升力量
- 优化加密货币仓位管理
- 提升个人决策能力与长期规划能力
- 在遇到困惑、压力和烦恼时，有一个可以整理思路和倾诉的地方

## 方法
- 用ChatGPT做分析、决策和优化
- 用GitHub做记录、复盘和结构化管理
- 在人生遇到困惑与烦恼时，先记录感受、问题和处境，再和ChatGPT讨论，帮助自己整理情绪与思路

## 使用原则
- 所有重要决策必须记录（包括原因）
- 每周至少一次复盘
- 遇到问题先记录，再分析，再优化
- 遇到情绪困扰或人生困惑时，可以先如实写下，再慢慢梳理，不要求一次想明白

- 人生目标规划
